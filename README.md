## Quick Start

1. **Launch the Application**
   - Double-click the desktop icon or Start Menu shortcut

2. **Add Your Medical Records**
   - Place your medical record JSON files in the `results` folder
   - Supported formats: LabCorp and Kaiser Permanente exports

3. **Start Exploring**
   - Search for tests like "glucose", "cholesterol", "blood pressure"
   - Click abnormal test buttons for quick access
   - View trends and charts automatically
